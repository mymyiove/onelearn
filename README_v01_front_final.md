## 반영 사항

- 대시보드 상단 통계 카드 순서: 마감 임박 / 추천 과정 / 진행 중 / 수료 완료
- 기본 필터 로직: 마감임박 과정이 있으면 마감임박, 없으면 전체
- 필터 탭 순서: 마감임박 / 전체 / 필수 / 선택 / 진행 중 / 미수료 / 수료 / 마감
- 문구 변경: 추천 과정을 확인하거나, 이어서 학습 해보세요!
- 필터별 빈 상태 문구 분기 처리
- 과정 카드 도넛 색상 보정: 수료 초록, 마감 빨강 계열
- 플레이어 기능 축약 없음: 사용법, 도움말, 학습 로그, 본인확인, 툴팁, 키보드 단축키, 화면이탈 감지, 전체화면/회전 보조 유지
- 대시보드/플레이어 모바일 푸터 포함 및 정렬 보정
