# OneLearn v01 Front Final Last Full Code

이번 수정 원칙:
- 마지막 기준본에서 요청 영역만 수정
- 대시보드/플레이어 공통 안정 규칙 유지
- 모바일 가로 초과 방지

반영 사항:
1. 모바일 푸터 복구: 고정 아님, 화면 하단 자연 배치. 로고 왼쪽/문구 오른쪽.
2. 모바일 명언창 고정: 2줄 clamp, 가로 찢어짐 방지.
3. 모바일 검색창 정리: 검색/초기화 버튼 크기와 폰트 통일.
4. 필터 뱃지를 검색창 아래로 이동.
5. 압축 → 목록 변경 및 목록형 과정 리스트 구현.
6. 공지사항 카드 높이 고정 및 3줄 초과 말줄임.
7. 대시보드 사용법 팝업을 플레이어 스타일로 재정리, 다음/이전 동작 보정.
8. 빠른검색 모달 문구/간격/결과 레이아웃 재정리.
9. 플레이어 자세히/접기 버튼 동일 스타일 유지.
10. 플레이어 CC 폰트 두께/크기 완화.
11. 플레이어 모바일 영상 가로 초과 방지 및 뱃지 가로 스크롤 유지.
