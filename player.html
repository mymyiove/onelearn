재생 문제 해결 가이드

재생에 문제가 있나요?

1. 브라우저 확인

최신 Chrome 또는 Edge 사용을 권장합니다.

2. 사내망/VPN/보안 프로그램 확인

사내망, VPN, 보안 프로그램이 영상 도메인을 차단할 수 있습니다.

3. 새로고침 후 재생

브라우저 새로고침 후 다시 재생해보세요.

4. 음량과 출력 장치 확인

브라우저 탭 음소거, OS 볼륨, 이어폰/스피커 연결 상태를 확인하세요.

5. 영상 끊김 확인

영상이 끊기면 다른 탭/프로그램을 종료하고 네트워크를 확인하세요.

6. 전체화면 권한 확인

전체화면이 안 되면 브라우저 권한 또는 팝업/보안 정책을 확인하세요.

7. 모바일 재생 환경 확인

모바일에서는 저전력 모드, 데이터 절약 모드, 백그라운드 앱을 끄면 안정적입니다.

8. 문의 시 필요한 정보

계속 문제가 있으면 과정명, 챕터명, 브라우저명, 발생 시간을 함께 전달해주세요.

위 가이드로도 해결이 안 되면 jhj11@wjthinkbig.com 으로 메일 주세요.
